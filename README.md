# DMZ-lab
Creación de una DMZ (Demilitarized Zone o Zona Desmilitarizada) con CISCO Packet Tracer

## Objetivo:
 El propósito del siguiente laboratorio es la configuración de una red con topología de **Estrella**, aplicando políticas de seguridad estrictas a través de **Listas de Control de Acceso (ACLs)** para disminuir la zona de vulnerabilidades, impidiendo intrusiones y movimientos laterales.
