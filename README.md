# DMZ-lab
Creación de una DMZ (Demilitarized Zone o Zona Desmilitarizada) con CISCO Packet Tracer

## Objetivo:
 El propósito del siguiente laboratorio es la configuración de una red con topología de **Estrella**, aplicando políticas de seguridad estrictas a través de **Listas de Control de Acceso (ACLs)** para disminuir la zona de vulnerabilidades, impidiendo intrusiones y movimientos laterales.

## Contenido del Repositorio

 El repositorio está organizado de la siguiente manera:

  - **`DMZ_PROJECT.pka`**: Archivo final de *Cisco Packet Tracer* con la topología completa de red y las reglas (ACLs) configuradas correctamente.
  - **`Informe/`**:
      - `Informe_DMZ_Lab.md`: Informe técnico del laboratorio, incluyendo la lógica de seguridad y configuraciones aplicadas.
  - **`Evidencias/`**:
      - Capturas de pantalla que muestran el éxito de las pruebas de conectividad:
        - Acceso HTTP desde la red externa.
        - Bloqueo de ICMP (ping) desde el exterior.
        - Aislamiento de la DMZ hacia la LAN interna mediante tráfico **established**.
